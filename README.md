启动命令：npm run serve 或者其它
端口：8081 或者自己配置

# frontend
技术栈：vue2（部分页面采用vue3写法）、vue-cli、element-plus、axios、TypeScript、vuex、vue-router
启动命令：npm run serve 或者其它
端口：8082 或者自己配置

启动命令：npm run start 或者其它

端口：3000 或者自己配置
