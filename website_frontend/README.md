# vue3-demo

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### 推送 dist 文件夹到 gh-pages 分支
```
git subtree push --prefix dist github gh-pages
```
